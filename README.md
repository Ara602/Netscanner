# 网络主机扫描软件

## 项目介绍
网络主机存活扫描软件，支持图形界面和命令行两种操作模式。可以扫描局域网内的主机，检测其在线状态，并获取主机名、MAC地址等信息。

## 功能特点
- 支持多种目标输入格式：单个IP、IP范围、CIDR网段
- 多线程并发扫描，提高扫描效率
- 图形化界面，直观显示扫描结果
- 命令行界面，便于脚本集成
- 支持结果导出：CSV、JSON、TXT、Excel格式
- 实时显示扫描进度和统计信息

## 系统要求
- Python 3.8+
- Windows/Linux/macOS
