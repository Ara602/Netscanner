一、软件简介
------------
本软件是一款网络主机存活扫描工具，能够快速扫描局域网内的在线主机，
并获取主机名、MAC地址、响应时间等信息。

作者：张夏灵
班级：网云2302
学号：542307280233

二、系统要求
------------
1. Windows 7/8/10/11（64位）
2. 需要管理员权限以获取MAC地址
3. 需要网络连接

三、使用方法
------------

1. 快速启动：
   双击 "网络主机扫描器.exe" 或 "启动软件.bat"

2. 扫描目标格式：
   - 单个IP: 192.168.1.1
   - IP范围: 192.168.1.1-100
   - 网段: 192.168.1.0/24
   - 自动检测: 快速扫描当前网络

3. 操作步骤：
   a. 输入扫描目标
   b. 设置线程数（默认100）
   c. 设置超时时间（默认2秒）
   d. 点击"开始扫描"
   e. 查看结果，可导出为CSV/JSON/TXT/Excel

四、功能特点
------------
✓ 图形化界面，操作简单
✓ 多线程扫描，速度快
✓ 支持多种扫描格式
✓ 结果导出功能
✓ 实时进度显示
✓ 统计信息展示

五、注意事项
------------
1. 扫描大量IP时可能会被防火墙拦截
2. 获取MAC地址需要管理员权限
3. 扫描速度受网络状况影响
4. 请勿用于非法用途
