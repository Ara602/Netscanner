## 1. 软件简介

### 1.1 软件概述
本软件是一款基于Python开发的网络主机存活扫描工具，支持图形界面和命令行两种操作模式。

### 1.2 主要功能
- 网络主机存活检测
- 主机信息收集（IP、MAC、主机名等）
- 多线程并发扫描
- 多种结果导出格式

### 1.3 系统要求
- 操作系统：Windows 7/10/11，Linux，macOS
- Python版本：3.8+
- 内存：至少512MB
- 磁盘空间：至少100MB

## 2. 安装指南

### 2.1 Python环境安装
1. 访问Python官网：https://www.python.org
2. 下载Python 3.8或更高版本
3. 安装时勾选"Add Python to PATH"

### 2.2 软件安装
```bash
# 方法一：从源码安装
git clone <项目地址>
cd 网络主机扫描软件
pip install -r requirements.txt

# 方法二：直接运行（无需安装）
python main.py
