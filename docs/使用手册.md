## 1. 软件简介

### 1.1 软件概述
本软件是一款基于Python开发的网络主机存活扫描工具，支持图形界面和命令行两种操作模式。

### 1.2 主要功能
- 网络主机存活检测
- 主机信息收集（IP、MAC、主机名等）
- 多线程并发扫描
- 多种结果导出格式

### 1.3 系统要求
- 操作系统：Windows 7/10/11，Linux，macOS
- Python版本：3.8+
- 内存：至少512MB
- 磁盘空间：至少100MB

## 2. 安装指南

### 2.1 Python环境安装
1. 访问Python官网：https://www.python.org
2. 下载Python 3.8或更高版本
3. 安装时勾选"Add Python to PATH"


### 2.2 软件安装
```bash
git clone https://github.com/Ara602/Netscanner.git
cd Netscanner
```

### 2.3 安装依赖
```bash
pip install -r requirements.txt
```

### 2.4 打包（可选）

#### 2.4.1 激活虚拟环境
```bash
# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate
```

#### 2.4.2 安装打包工具
```bash
pip install pyinstaller
pip install pyinstaller-hooks-contrib  # 额外的hooks支持
```

#### 2.4.3 执行打包

**Windows系统：**
```bash

**Linux/macOS系统：**
```bash

```

## 4. 功能详解
### 4.1 扫描模式说明
### 4.1.1 单IP扫描
功能：扫描指定单个IP地址
适用场景：检查特定主机是否在线
```bash
#示例：
python main.py cli 192.168.1.100
```

### 4.1.2 IP范围扫描
功能：扫描连续IP地址范围
格式：起始IP-结束IP（如192.168.1.1-254）
```bash
#示例：
python main.py cli 192.168.1.1-100
```

### 4.1.3 CIDR网段扫描
功能：扫描整个子网
格式：CIDR表示法（如192.168.1.0/24）
```bash
#示例：
python main.py cli 192.168.1.0/24
```
### 4.2 参数配置详解
### 4.2.1 线程数控制 (-t/--threads)
作用：控制并发扫描线程数量
默认值：50个线程
建议：
局域网环境：100-200线程
互联网扫描：20-50线程
性能较差设备：10-30线程
```bash
#示例：
python main.py cli 192.168.1.0/24 -t 100
```
#### 4.2.2 超时设置 (--timeout)
作用：设置每个主机的响应超时时间
默认值：2秒
单位：秒
建议：
局域网：1-2秒
远程网络：3-5秒
```bash
#示例：
python main.py cli 192.168.1.0/24 --timeout 3
```
#### 4.2.3 端口扫描功能(可选)
端口范围指定：-p 80,443 或 -p 1-1000
常用端口扫描：--common-ports
全端口扫描：-p 1-65535

### 4.3 输出格式选项
#### 4.3.1 文本输出
```bash
# 控制台显示结果
python main.py cli 192.168.1.0/24

# 保存为文本文件
python main.py cli 192.168.1.0/24 -o result.txt --format txt
```

#### 4.3.2 CSV格式
```bash
# 保存为CSV文件
python main.py cli 192.168.1.0/24 -o result.csv --format csv
4.3.3 Excel格式

#### 4.3.3 Excel格式
```bash
# 保存为Excel文件（需安装pandas库）
python main.py cli 192.168.1.0/24 -o result.xlsx --format excel
```

#### 4.3.4 JSON格式
```bash
# 保存为JSON文件，便于程序处理
python main.py cli 192.168.1.0/24 -o result.json --format json
```

### 4.4 高级功能
#### 4.4.1 静默模式
```bash
# 只输出存活主机，不显示详细信息
python main.py cli 192.168.1.0/24 --quiet

# 仅输出IP列表，适合脚本处理
python main.py cli 192.168.1.0/24 --simple
```

#### 4.4.2 详细模式
```bash
# 显示详细扫描过程信息
python main.py cli 192.168.1.0/24 --verbose

# 调试模式，显示所有细节
python main.py cli 192.168.1.0/24 --debug
```

#### 4.4.3 定时任务
```bash
# 定期扫描（如每30分钟扫描一次）
python main.py cli 192.168.1.0/24 --interval 1800

```
## 5. 常见问题（补充）
### 5.1 权限问题
问题描述：在Linux/macOS上运行时提示权限不足

```bash
#解决方案1：使用sudo权限运行
sudo python main.py cli 192.168.1.0/24

#解决方案2：使用Python虚拟环境
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
### 5.2 依赖安装失败
问题描述：pip install时出现错误

```bash
# 更新pip工具
python -m pip install --upgrade pip

# 使用国内镜像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 单独安装失败包
pip install 包名 --no-deps
```

## 5.3 扫描结果不准确
问题描述：部分主机在线但未扫描到

```bash
# 增加超时时间
python main.py cli 192.168.1.0/24 --timeout 5

# 减少线程数，避免网络拥堵
python main.py cli 192.168.1.0/24 -t 20

# 使用ping扫描替代端口扫描（如果支持）
python main.py cli 192.168.1.0/24 --ping-only
```
## 5.4 软件启动失败
问题描述：双击exe文件无反应或报错

```bash
#解决方案1：使用命令行查看错误信息

#解决方案3：检查系统兼容性

## 5.5 内存占用过高
问题描述：扫描大型网络时内存占用飙升

```bash
# 限制扫描范围
python main.py cli 192.168.1.1-100

# 分批扫描
python main.py cli 192.168.1.0/24 --batch-size 50

# 调整线程数
python main.py cli 192.168.1.0/24 -t 30
```
## 5.6 图形界面卡顿
问题描述：GUI界面响应缓慢

```bash
# 减少实时刷新频率
在设置中调整"刷新间隔"为500ms或更高

# 关闭不必要的动画效果
在设置中禁用"扫描动画"和"实时图表"

# 使用轻量模式
python main.py gui --light-mode
```
